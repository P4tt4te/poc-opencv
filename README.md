# OpenCV sandbox

This is a proof of concept project to demonstrate the use of OpenCV with C++.

## Dependencies

- **OpenCV 4.10.0**
- **FMT 11.0.2**
- **CVUI 2.7.0** (Integrated GUI for OpenCV)
- **vcpkg** (Package manager)

## Requirements

- **CMake** (3.25+)
- **Ninja**
- **Visual Studio 2022/2026** (MSVC v143 toolset + vcpkg extension)
- **A C++20 compatible compiler**

## Usage

After building the project, you can run the generated executable :
**poc-opencv.exe**
